# Jordan Leon – Fitness Coaching Website

One-Page-Website für das Personal Fitness Coaching von [Jordan Leon](https://www.instagram.com/jordan__leon/).

## Stack

Reines HTML/CSS/JavaScript – kein Build-Tool, keine Dependencies. Kann direkt auf jedem statischen Hosting (GitHub Pages, Netlify, IONOS, …) deployt werden.

## Struktur

```
index.html        # Landing Page (Hero, Über mich, Coaching, Ablauf, FAQ, CTA)
impressum.html    # Impressum (Platzhalter – Daten eintragen!)
datenschutz.html  # Datenschutzerklärung (Platzhalter – prüfen lassen!)
css/styles.css    # Design (Dark Theme, rote Akzente – passend zum Flyer)
js/main.js        # Mobile-Menü, Sticky-Nav, Scroll-Animationen
assets/           # Bilder (Coaching-Flyer)
```

## Lokal ansehen

`index.html` einfach im Browser öffnen, oder:

```
npx serve .
```
